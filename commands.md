# Commands

**To begin Solana Raffle AI Bot and access the Main Menu:**

```bash
  Type /start 
```

**To obtain your chat ID:**

```bash
  Type /chatid
```

**For information about Solana Raffle AI Bot:**

```bash
  Type /info
```

**For FAQ:**

```bash
  Type /faq
```

**For leaderboard results:**

```bash
  Type /leaderboard
```
